# Skill: Marketing

## When to Use This Skill

Use this skill for tasks with `type: marketing`. Marketing tasks involve creating content, devising growth strategies, writing copy, or analyzing audiences to grow the project's reach and adoption.

## Marketing Process

### Step 1 — Understand the Product & Audience
Before writing anything:
- Read `/context/project/mission.md` for the core value proposition
- Review existing results in `/results/` for prior work
- Identify: Who is the target audience? What problem do we solve for them?

### Step 2 — Define the Goal
What does success look like for this specific task?
- More GitHub stars?
- Developer signups?
- Community contributions?
- Partnership inquiries?

### Step 3 — Create the Content
Match tone to platform:

| Platform | Tone | Format |
|---|---|---|
| GitHub README | Technical, direct | Headers, code blocks, badges |
| Twitter/X | Punchy, specific | 280 chars, hook first |
| Hacker News | Honest, technical | "Show HN:" with context |
| Reddit (r/programming) | Educational, humble | Discussion starter |
| Product Hunt | Enthusiastic, benefit-focused | Problem → Solution → Call to action |
| Dev.to / Hashnode | Detailed, tutorial-style | Long-form with examples |

### Step 4 — Structure the Output
Save results at `results/{task_name}.md`:

```markdown
# Marketing: {title}

**Date:** {YYYY-MM-DD}
**Task:** {task_filename}
**Goal:** {specific goal}

## Audience Analysis
- Primary audience: ...
- Pain points: ...
- Where they hang out: ...

## Content Drafts

### Twitter/X Thread
Tweet 1: ...
Tweet 2: ...

### GitHub README Section
...

### Hacker News Post
Title: ...
Body: ...

### Product Hunt Launch Copy
Tagline: ...
Description: ...

## Growth Strategy
1. ...
2. ...

## Metrics to Track
- ...
```

### Step 5 — Update Learnings
Append insights about what messaging or channels might work to `/context/memory/learnings.md`.

## Quality Checklist

- [ ] Value proposition is clear in the first sentence
- [ ] Content is specific (concrete benefits, not vague claims)
- [ ] Multiple channel drafts included
- [ ] Audience analysis completed
- [ ] Result file saved to `results/{task_name}.md`
- [ ] Task file status updated to `completed`
- [ ] Learnings appended to `context/memory/learnings.md`
